# Analizador de Gastos

Practicando Git y GitHub. Un pequeño analizador de gastos que lee archivos CSV y muestra estadísticas.

## ¿Cómo ejecutarlo?

```bash
python analizador.py
```

## ¿Qué muestra?
Total gastado: $11500
Promedio por gasto: $1437.5
Gasto mayor: supermercado ($3500)
Categoría con más gastos: comida

## ¿Cómo correr los tests?

```bash
python -m pytest test_analizador.py -v
```
